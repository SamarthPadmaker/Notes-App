## 📝 Notes Web App (MERN Stack)

## 📌 Overview
A full-stack Notes application with authentication and CRUD functionality.

## 🚀 Features
- Signup & Login
- JWT Authentication
- Protected Routes
- CRUD Operations
- Search Notes

## 🛠️ Tech Stack
- Frontend: React.js, Tailwind CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Auth: JWT, bcrypt
